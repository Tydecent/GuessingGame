# GuessingGame

修订日期：2026/02/06

## 概述
一个非常简单的猜数字游戏，但是我们计划加上许多神奇的功能。

## 可选启动参数
`-debug`：启用调试模式（保留，调试模式开发中）

`-num custom_number`：自定义数字

`-version`：查询版本

## 下载源代码
使用以下命令克隆仓库：
```bash
git clone https://github.com/Tydecent/GuessingGame.git
```
