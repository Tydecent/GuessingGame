# 贡献指南

感谢您有兴趣为GuessingGame做出贡献！

## 如何贡献

1. 在提交代码之前，请先为重大更改和新功能[**创建一个Issue**](https://github.com/Tydecent/GuessingGame/issues/new)**以进行讨论**；
2. Frok本仓库；
3. 创建你的功能分支 (`git checkout -b feature/amazing-feature`)。
4. 提交你的更改 (`git commit -m 'feat: add some amazing feature'`)。
5. 推送到分支 (`git push origin feature/amazing-feature`)。
6. 开启一个 Pull Request。

## 报告问题

请[在Issue页面报告问题](https://github.com/Tydecent/GuessingGame/issues/new)，报告问题时请尽可能地包含：
- 清晰的问题描述；
- 复现步骤；
- 期望行为与实际行为；
- 相关截图。

*我们计划在未来为程序推出日志功能，到那时，你还可以上传日志。*

## 代码规范
请使用`cargo fmt`统一风格并且`cargo check`命令检查通过。

## 许可证
通过提交贡献，您同意您的作品将根据本项目的[MIT许可证](LICENSE)进行授权
